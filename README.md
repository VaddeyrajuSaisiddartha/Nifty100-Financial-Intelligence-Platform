## Live Application

🌐 Live Demo: https://nifty-100-intelligence-549099183935.asia-southeast1.run.app

## Project Overview

Nifty100 Financial Intelligence Platform is a full-stack financial analytics system designed to provide insights into Nifty100 companies through interactive dashboards, investor analytics, sector analysis, and AI-powered financial intelligence.

The platform combines data warehousing, ETL pipelines, business intelligence dashboards, and modern web technologies to help users analyze company performance and market trends.

---

## Features

### Executive Overview Dashboard

* Total Companies Analysis
* Average ROE Analysis
* Average ROCE Analysis
* Top Performing Companies
* Sector Distribution Analysis

### Financial Trends Dashboard

* Operating Profit Trends
* Net Profit Trends
* EPS Analysis
* Financial Performance Tracking
* Company-wise Filtering

### Sector Analytics Dashboard

* Sector Distribution Analysis
* Company Count by Sector
* Average Book Value Analysis
* Top Company Performance Table
* Sector-wise Insights

### Investor Insights Dashboard

* Highest ROE Companies
* Highest ROCE Companies
* Highest Book Value Companies
* Top 10 Company Rankings
* Investor Comparison Analytics
