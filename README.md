# 📖 Project Overview

The **Nifty100 Financial Intelligence Platform** is a full-stack fintech analytics application developed during an internship at **BlueStock Fintech**. The platform provides investors, analysts, and researchers with comprehensive insights into Nifty100 companies through interactive dashboards, financial analytics, sector analysis, and investor intelligence.

The system integrates:

* Data Warehousing
* ETL Pipelines
* Business Intelligence Dashboards
* Financial Analytics
* Investor Insights
* Cloud Deployment
* REST APIs
* Interactive Web Application

NIFTY 100 represents the top 100 large-cap companies listed on the National Stock Exchange (NSE) and covers major sectors of the Indian economy.

---

# 🎯 Project Objectives

* Build a Financial Intelligence Platform for Nifty100 companies.
* Develop ETL pipelines for financial data processing.
* Create a PostgreSQL Data Warehouse.
* Design REST APIs using Django.
* Build a modern React-based frontend.
* Create Power BI dashboards for analytics.
* Deploy the platform to the cloud.
* Provide actionable investor insights.

# ⚙️ ETL Pipeline

The ETL process consists of:

### Extract

Financial datasets are collected from structured sources.

### Transform

Data is cleaned, normalized, validated, and standardized.

### Load

Processed data is loaded into PostgreSQL Data Warehouse for reporting and analytics.
